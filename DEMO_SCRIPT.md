# CasperGuard Demo Video Script

**Target length:** 3-4 minutes
**Tool:** Screen record with QuickTime (Cmd+Shift+5) or Loom
**Resolution:** 1920x1080 recommended
**URL:** https://casper-guard.vercel.app

---

## Scene 1: Landing Page (0:00 - 0:40)

**Open:** https://casper-guard.vercel.app

**Narrate (or add text overlay):**
> "CasperGuard is an AI-powered DeFi insurance protocol built on Casper Network. It uses 4 autonomous AI agents to protect your DeFi positions — all powered by the Casper AI Toolkit."

**Actions:**
1. Let the landing page animations play (particles, fade-ins)
2. Slowly scroll down to show:
   - The stats bar (4 agents, 6 protocols, x402 payments)
   - Agent Fleet cards (RiskSentinel, ClaimBot, UnderwriteAI, VaultKeeper)
   - Casper AI Toolkit integration section (x402, MCP, CSPR.click, Odra)
   - How It Works flow
3. Click **"Launch App"** button

---

## Scene 2: Dashboard — Live Casper Data (0:40 - 1:20)

**Narrate:**
> "The dashboard pulls LIVE data from Casper Testnet. You can see the real block height, era, and deploy count updating in real-time from api.testnet.cspr.live."

**Actions:**
1. Point out the **LiveNetworkBar** at the top — real block number, era, deploys
2. Show the stat cards: TVL, Active Policies, Reserve Ratio, Agent Transactions
3. Scroll to the **Protocol Risk Trends chart** — explain the 3 monitored protocols
4. Point out the **Monitored Protocols** panel with the green "LIVE" badge
5. Show the **Live Agent Activity feed** — each agent's recent actions
6. Show the **x402 Payments** panel — micropayments between agents

---

## Scene 3: AI Agents Page (1:20 - 1:50)

**Actions:**
1. Click **"AI Agents"** in the sidebar
2. Point out the **Live Agent Status** banner showing the real block being scanned
3. Show the **Agent Architecture diagram** — how agents connect via MCP + x402
4. Click **"View Activity"** on RiskSentinel to expand its activity log
5. Show MCP connection badges on each agent card

**Narrate:**
> "Each agent connects to Casper Network through MCP Servers. RiskSentinel scans every block for vulnerabilities. ClaimBot processes claims with AI. UnderwriteAI prices risk dynamically. VaultKeeper manages reserves."

---

## Scene 4: Create a Policy — Live API (1:50 - 2:30)

**Actions:**
1. Click **"Policies"** in the sidebar
2. Show existing policies with risk scores and status badges
3. Click **"+ New Policy"** button
4. Fill the form:
   - Wallet: `demo.cspr`
   - Protocol: Select **NexusDEX** (high risk)
   - Coverage: `50000`
5. Point out the **AI Risk Assessment** preview (risk score, premium rate, x402 payment)
6. Click **"Create Policy & Pay Premium via x402"**
7. Show the **UnderwriteAI result** — policy ID, premium paid, on-chain TX hash, block height

**Narrate:**
> "When you create a policy, UnderwriteAI calculates the premium based on real-time risk. The transaction is anchored to an actual Casper Testnet block. Premium is paid via x402 micropayment protocol."

---

## Scene 5: Submit a Claim — AI Assessment (2:30 - 3:10)

**Actions:**
1. Click **"Claims"** in the sidebar
2. Show existing claims with AI assessment badges (APPROVE/REVIEW)
3. Click on **CLM-001** to show the full AI assessment detail:
   - Risk Score, Confidence, Decision
   - AI Reasoning
   - Risk Factors with progress bars
4. Close modal, then click **"+ Submit Claim"**
5. Fill the form:
   - Policy: Select any active policy
   - Amount: `10000`
   - Reason: `Smart contract vulnerability detected`
6. Click **"Submit Claim for AI Review"**
7. Show the **ClaimBot Assessment** result — AI decision, confidence, reasoning, TX hash

**Narrate:**
> "ClaimBot autonomously analyzes on-chain evidence, assesses risk factors, and makes a decision — all anchored to real Casper blockchain data. The assessment includes confidence scores and detailed reasoning."

---

## Scene 6: Closing (3:10 - 3:30)

**Actions:**
1. Navigate back to the landing page
2. Scroll to the Casper AI Toolkit section

**Narrate:**
> "CasperGuard demonstrates the full Casper AI Toolkit — MCP Servers for blockchain queries, x402 for agent-to-agent micropayments, CSPR.click Agent Skill for wallet interactions, and Odra smart contracts for on-chain logic. Built for the Casper Agentic Buildathon 2026."

## Tips for Recording

- **Use desktop view** (1280px+ width) for best appearance
- **Move your mouse slowly** so viewers can follow
- **Pause 1-2 seconds** on important elements
- **Close other browser tabs** to keep it clean
- Record in a quiet room if narrating
- If not narrating, add text overlays in post-production (iMovie, CapCut, etc.)
- Upload to YouTube as **Unlisted** and include link in DoraHacks submission
