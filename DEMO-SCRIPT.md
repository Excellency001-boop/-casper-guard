# CasperGuard Demo Video Script

**Target length:** 3-5 minutes
**Recording tool:** Use QuickTime (Cmd+Shift+5) or Loom (free screen recorder)
**Resolution:** 1920x1080 recommended
**URL:** https://casper-guard.vercel.app

---

## INTRO (0:00 - 0:20)

**Open the landing page at https://casper-guard.vercel.app**

> "This is CasperGuard — an AI-powered DeFi insurance protocol built on Casper Network. It uses 4 autonomous AI agents to protect DeFi positions, leveraging the full Casper AI Toolkit including MCP Servers, x402 Micropayments, CSPR.click Agent Skill, and Odra Smart Contracts."

- Show the animated landing page
- Scroll slowly through the hero, stats bar, and agent fleet section
- Point out the "Built with Casper AI Toolkit" section

---

## LIVE NETWORK DATA (0:20 - 0:45)

**Click "Launch Dashboard"**

> "The entire platform runs on live Casper Testnet data. You can see real block heights, era numbers, and deploy counts pulled directly from api.testnet.cspr.live."

- Point out the LiveNetworkBar at the top showing real block height and era
- Show the "All systems operational" indicator

---

## DASHBOARD (0:45 - 1:30)

> "The dashboard shows the insurance vault's key metrics — Total Value Locked, active policies, reserve ratio, and autonomous agent transactions."

- Show the 4 stat cards
- Point to the Protocol Risk Trends chart

> "Our RiskSentinel AI agent continuously monitors 6 DeFi protocols on Casper. Notice the LIVE badge — these risk scores update in real-time using data from the Casper MCP Server."

- Point to the Monitored Protocols panel with the LIVE badge
- Show the risk scores (LOW, MEDIUM, HIGH, CRITICAL)

> "The Live Agent Activity feed shows what each AI agent is doing — processing claims, adjusting premiums, rebalancing vaults — all autonomously."

- Scroll the agent activity feed
- Point to the x402 Payments section

---

## AI AGENTS (1:30 - 2:15)

**Click "AI Agents" in sidebar**

> "Here's our fleet of 4 autonomous agents, all connected to the Casper Network via MCP."

- Show the architecture diagram
- Point out each agent card: RiskSentinel, ClaimBot, UnderwriteAI, VaultKeeper

> "Each agent has its own MCP connections and performs specialized tasks. Let me expand one to show its activity."

- Click "View Activity" on RiskSentinel
- Show the activity log with transaction hashes

> "Notice the live status banner — all agents are scanning the current block in real-time."

---

## POLICY CREATION (2:15 - 3:00)

**Click "Policies" in sidebar**

> "Users can browse existing policies or create new ones. Each policy is underwritten by our UnderwriteAI agent."

- Show the policy cards with risk scores
- Click filter tabs (Active, Claimed, etc.)

**Click "New Policy"**

> "Let me create a new policy. I'll insure against NexusDEX — notice how the AI instantly calculates the risk score and premium rate."

- Type a wallet name (e.g., "demo.cspr")
- Select "NexusDEX" from the protocol dropdown
- Enter coverage amount (e.g., "50000")
- Show the AI Risk Assessment panel that appears
- Click "Create Policy & Pay Premium via x402"

> "UnderwriteAI processes the request, calculates the premium, and creates the policy with an on-chain transaction — premium paid via x402 micropayment."

- Show the result modal with policy ID, TX hash, block height

---

## CLAIMS PROCESSING (3:00 - 3:45)

**Click "Claims" in sidebar**

> "When an incident occurs, users submit claims which are processed by ClaimBot — our AI claims assessor."

- Show existing claims with their AI assessments
- Click on CLM-001 to show the detail modal

> "ClaimBot analyzes on-chain evidence, calculates risk factors like evidence strength and protocol audit status, then makes an autonomous approve/reject decision with confidence scores."

- Show the AI Assessment section with risk score, confidence, reasoning
- Show the risk factors with progress bars

**Click "Submit Claim"**

> "Let me submit a new claim. ClaimBot will analyze the evidence in real-time."

- Select a policy, enter amount, reason, and evidence
- Click Submit
- Show the live AI assessment result

---

## WRAP UP (3:45 - 4:15)

**Navigate back to landing page**

> "CasperGuard demonstrates the full power of the Casper AI Toolkit:
> - MCP Servers for real-time blockchain data
> - x402 Protocol for agent-to-agent micropayments
> - CSPR.click Agent Skill for on-chain actions
> - Odra Framework for smart contract interactions
>
> All 4 agents work autonomously, protecting DeFi users 24/7 on Casper Network."

- End on the landing page hero

## TIPS

1. Record in a quiet environment
2. Use a clean browser (no bookmarks bar, no other tabs)
3. Go fullscreen or use a clean window
4. Speak clearly and at moderate pace
5. If using Loom: it adds a webcam bubble which looks professional
6. Upload the video to YouTube (unlisted) and link in DoraHacks submission
